# Label-IT

Label-IT is an interactive educational game designed to help students learn the names and functions of tools by dragging and dropping labels onto images.

## How to Play
1. Click **START**
2. Read the tutorial
3. Select a level
4. Drag labels onto the correct parts of the tool
5. Click **Check Answers**
6. Answer everthing correctly to complete the level to unlock the next one

## Features
- Drag-and-drop labeling
- Multiple levels
- Progression-based unlocking
- Audio controls
- Designed for classroom use

## Technologies Used
- HTML
- CSS
- JavaScript

## Deployment
This project is deployed as a static website and runs entirely in the browser.
